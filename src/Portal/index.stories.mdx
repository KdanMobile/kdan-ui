# Portal

The portal component renders its children into a new "subtree" outside of current DOM hierarchy.

## Import

```js
import { Portal } from "@kdanmobile/kdan-ui";

const App = () => (
  <Portal>
    <div />
  </Portal>
);
```

## Props

|   Props   | Type                  | Default |
| :-------: | :-------------------- | :------ |
| children  | `React.ReactNode`     |         |
| container | `Element or Function` |         |
